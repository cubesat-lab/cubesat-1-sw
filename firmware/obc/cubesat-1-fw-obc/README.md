# cubesat-1-fw-obc

**CubeSat-1 Firmware for OBC** is an executable Rust project developed for OBC system of the [**CubeSat-1**](https://example.com/) project.

### Compiling

- Compile OBC Firmware

    ```bash
    cd ./firmware/obc/cubesat-1-fw-obc/
    cargo build
    ```
